# 模板库管理系统 - 测试用例文档

## 测试概述

本测试用例文档基于 `PRD_template-library.md` 和 `DESIGN_template-library.md` 设计文档，旨在验证模板库管理系统的所有功能是否正常运行。

## 测试环境

- **前端环境**: Vue 3 + Vite 4.5+
- **后端环境**: Spring Boot 3.2.0
- **数据库**: MySQL 8.0+
- **浏览器**: Chrome 90+、Firefox 88+、Safari 14+

## 测试用例

### 1. 模板管理功能测试

#### 1.1 模板列表展示

**测试ID**: TC-001
**测试类型**: 功能测试
**测试目的**: 验证模板列表是否正确展示
**测试步骤**:
1. 登录系统
2. 进入模板库管理页面
3. 查看模板列表

**预期结果**:
- 模板列表正确显示所有模板
- 模板信息完整（名称、描述、分类、创建时间等）
- 分页功能正常

#### 1.2 模板新增

**测试ID**: TC-002
**测试类型**: 功能测试
**测试目的**: 验证模板新增功能是否正常
**测试步骤**:
1. 登录系统
2. 进入模板库管理页面
3. 点击「新增模板」按钮
4. 填写模板信息
5. 上传模板文件
6. 点击「保存」按钮

**预期结果**:
- 模板成功新增
- 页面跳转到模板列表
- 新模板显示在列表中

#### 1.3 模板编辑

**测试ID**: TC-003
**测试类型**: 功能测试
**测试目的**: 验证模板编辑功能是否正常
**测试步骤**:
1. 登录系统
2. 进入模板库管理页面
3. 选择一个模板，点击「编辑」按钮
4. 修改模板信息
5. 点击「保存」按钮

**预期结果**:
- 模板信息成功更新
- 页面跳转到模板列表
- 模板列表中显示更新后的信息

#### 1.4 模板删除

**测试ID**: TC-004
**测试类型**: 功能测试
**测试目的**: 验证模板删除功能是否正常
**测试步骤**:
1. 登录系统
2. 进入模板库管理页面
3. 选择一个模板，点击「删除」按钮
4. 在确认弹窗中点击「确定」

**预期结果**:
- 模板成功删除
- 模板从列表中消失
- 显示删除成功提示
