## 环境准备要求

### 1. 硬件要求

- **服务器**: 至少 2 核 CPU，4GB 内存，50GB 存储空间
- **网络**: 稳定的网络连接，带宽至少 10Mbps

### 2. 软件要求

- **Java**: JDK 17 或更高版本
- **Node.js**: 14.x 或更高版本
- **MySQL**: 8.0 或更高版本
- **Maven**: 3.6 或更高版本
- **Web 服务器**: Nginx 1.18 或更高版本（用于部署前端）

### 3. 环境变量

| 环境变量 | 说明 | 默认值 | 备注 |
|----------|------|--------|------|
| `DB_HOST` | 数据库主机地址 | localhost |  |
| `DB_PORT` | 数据库端口 | 3306 |  |
| `DB_NAME` | 数据库名称 | pm_portfolio_hub |  |
| `DB_USER` | 数据库用户名 | root |  |
| `DB_PASSWORD` | 数据库密码 |  | 必须设置 |
| `UPLOAD_PATH` | 文件上传路径 | ./uploads |  |
| `SERVER_PORT` | 后端服务端口 | 8080 |  |
| `FRONTEND_URL` | 前端地址 | http://localhost:3001 |  |

## 上线操作步骤

### 1. 后端部署

1. **构建项目**
   ```bash
   cd backend
   mvn clean package
   ```

2. **配置环境变量**
   ```bash
   # 创建 .env 文件
   echo "DB_HOST=localhost
   DB_PORT=3306
   DB_NAME=pm_portfolio_hub
   DB_USER=root
   DB_PASSWORD=your_password
   UPLOAD_PATH=./uploads
   SERVER_PORT=8080
   FRONTEND_URL=http://localhost:3001" > .env
   ```

3. **启动服务**
   ```bash
   java -jar target/hub-1.0.0.jar
   ```

### 2. 前端部署

1. **构建项目**
   ```bash
   cd frontend
   npm install
   npm run build
   ```

2. **配置 API 基础 URL**
   ```bash
   # 修改 api.js 中的 baseURL
   sed -i 's|http://localhost:8080|http://your-server:8080|g' dist/assets/*.js
   ```

3. **部署到 Nginx**
   ```bash
   # 复制构建产物到 Nginx 目录
   cp -r dist/* /usr/share/nginx/html/
   
   # 配置 Nginx
   echo 'server {
       listen 80;
       server_name your-domain.com;
       
       location / {
           root /usr/share/nginx/html;
           index index.html;
           try_files $uri $uri/ /index.html;
       }
       
       location /api {
           proxy_pass http://localhost:8080;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
       }
   }' > /etc/nginx/conf.d/pm-portfolio.conf
   
   # 重启 Nginx
   systemctl restart nginx
   ```

### 3. 数据库初始化

1. **创建数据库**
   ```sql
   CREATE DATABASE IF NOT EXISTS pm_portfolio_hub CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```

2. **执行数据库脚本**
   ```bash
   mysql -u root -p pm_portfolio_hub < schema.sql
   ```

### 4. 验证上线

1. **访问前端**
   - 打开浏览器，访问 http://your-domain.com
   - 登录系统，验证所有功能是否正常

2. **访问后端 API**
   - 访问 http://your-domain.com/api/swagger-ui.html
   - 验证 API 文档是否正常

3. **测试核心功能**
   - 测试模板上传、预览、编辑、删除等功能
   - 测试分类管理功能
   - 测试搜索与筛选功能
